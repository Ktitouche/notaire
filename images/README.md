# Images Guide for Notary Website

This folder should contain professional images for your notary website.

## Recommended Images

### 1. Hero/Header Images
- **hero-office.jpg** - Professional office space, signatures, or legal documents
  - Dimensions: 1920x1080px minimum
  - Use: Home page hero section background

### 2. About Page
- **notary-portrait.jpg** - Professional photo of Maître Pierre Durand
  - Dimensions: 800x1000px
  - Use: About page introduction section

- **team-office.jpg** - Team photo or office environment
  - Dimensions: 1200x800px
  - Use: About page team section

### 3. Service Icons (Optional)
If you prefer custom icons over Font Awesome:
- Real estate icon
- Succession icon
- Family law icon
- Corporate law icon
- Authentication icon

### 4. Office Photos
- **office-exterior.jpg** - Building exterior
- **office-interior.jpg** - Reception or meeting room
- **consultation-room.jpg** - Client consultation space

### 5. Document Icons
- **pdf-icon.png** - For downloadable documents section
  - Dimensions: 200x200px

## Image Guidelines

### Format
- **JPEG** for photos (compress to 80-85% quality)
- **PNG** for logos and icons with transparency
- **WebP** for modern browsers (optional optimization)

### Optimization
- Use tools like TinyPNG or ImageOptim to compress images
- Target file size: < 200KB for hero images, < 100KB for others
- Maintain aspect ratios to prevent distortion

### Stock Photo Sources (if needed)
- Unsplash.com (free, high-quality)
- Pexels.com (free)
- Pixabay.com (free)
- Adobe Stock (paid)
- Shutterstock (paid)

### Search Keywords for Stock Photos
- "notary office"
- "legal documents signing"
- "professional office"
- "law office interior"
- "business handshake"
- "legal consultation"
- "french notary" (for specific context)

## Current Image Placeholders

The website currently uses Unsplash URLs for demo images:
1. `https://images.unsplash.com/photo-1554224311-beee460c201f` - Office image
2. `https://images.unsplash.com/photo-1507679799987-c73779587ccf` - Portrait

Replace these with your own images by:
1. Saving images to this folder
2. Updating image src attributes in HTML files
3. Example: `<img src="images/hero-office.jpg" alt="Office">`

## Alt Text Recommendations

Always include descriptive alt text for accessibility:
- ✅ Good: "Maître Pierre Durand consulting with a client in the office"
- ❌ Bad: "image1" or empty alt=""

## Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `team-photo-2024.jpg`
- ✅ `office-exterior.jpg`
- ❌ `IMG_1234.jpg`
- ❌ `Photo1.JPG`

## Copyright

Ensure you have the rights to use all images:
- Own photography: ✅
- Purchased stock photos: ✅
- Free stock with attribution: ✅ (check license)
- Google Images: ❌ (unless verified license-free)
